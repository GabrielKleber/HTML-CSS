Aqui temos um termo em destaque usando a tag <strong> (tag semântica).

Itálico / Ênfase

Aqui temos um termo em itálico usando a tag <i> (tag não semântica).

Aqui temos um termo em ênfase usando a tag <em> (tag semântica).

Texto marcado

Podemos também criar um texto marcado usando a tag <mark>

E para mudar a cor do texto marcado se usa a tag <mark style="background-color: ">

Mas não muda a cor do marcador dos parágrafos seguintes

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag <ins> (sêmantica)

Existe também o sublinhado com a tag <u> (não sêmantica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usando a tag <sup> no 20

Texto subscrito

Para inserir coisas do tipo H2O usando a tag <sub> no 2
